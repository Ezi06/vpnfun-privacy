VPNFun connects to VPN servers to provide internet routing services, but we do not collect, store, or monitor any data from these connections. The servers are used solely for network routing purposes.

2. How We Use Your Information

Since we collect no data, there is no information to use. VPNFun connects to servers only for network routing purposes without collecting any user data.

3. Data Protection

3.1 Network Extensions

VPNFun uses Apple's built-in Network Extensions framework to provide VPN functionality. This leverages iOS's native networking capabilities for secure connections.

3.2 Zero Data Collection

We maintain a strict zero-data collection policy. No user data is stored, transmitted, or processed by our servers, even though we connect to VPN servers for routing.

3.3 Server Usage

VPN servers are used exclusively for internet traffic routing. No data collection, logging, or monitoring occurs on these servers.

4. Technical Implementation

VPNFun is built using:

  • iOS Network Extensions: Apple's native framework for VPN functionality
  • PacketTunnelProvider: iOS system component for handling network traffic
  • VPN Servers: Used for internet routing without data collection
  • Local Processing: All VPN operations happen within iOS system boundaries

This approach ensures maximum compatibility with iOS security features and system-level privacy protections while providing VPN routing services.
